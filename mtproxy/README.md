# A set of scripts for automated installation of various MTProto Proxy

[По-русски / In Russian](README.ru.md)

Here is a set of simple scripts for automated installation/removal of various MTProto Proxy.

- install_mtg.sh, uppdate_mtg.sh, remove_mtg.sh - Script for install, uppdate and remove [MTG Proxy](https://github.com/9seconds/mtg)
- install_telemt.sh, uppdate_telemt.sh, remove_telemt.sh - Script for install, uppdate and remove [TeleMT Proxy](https://github.com/telemt/telemt)
