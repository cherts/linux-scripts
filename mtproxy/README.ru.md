# Набор скриптов для автоматизированной установки различных MTProto Proxy

[In English / По-английски](README.md)

Здесь представлен набор простых скриптов для автоматизированной установки/удаления различных MTProto Proxy.

- install_mtg.sh, uppdate_mtg.sh, remove_mtg.sh - Скрипты установки, обновления и удаления [MTG Proxy](https://github.com/9seconds/mtg)
- install_telemt.sh, uppdate_telemt.sh, remove_telemt.sh - Скрипты установки, обновления и удаления [TeleMT Proxy](https://github.com/telemt/telemt)
